# Dotfiles


Personal Setup of the Dotfiles for my custom workflow


# Getting Startet

 - Install `xcode`
 - clone this repo

Install oh-my-zsh  
```
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```

run installation file 
```
./install.sh
```
